## 🍎 Mac 端启动

### 特性
- ✅ 自动检测运行环境
- ✅ 自动导入数据库（已导入则跳过）
- ✅ 自动安装前端依赖（已安装则跳过）
- ✅ 一键启动后端 + 管理后台 + 用户前台
- ✅ 打印所有角色的账号密码
- ✅ 实时聚合显示三个服务的日志
- ✅ Ctrl+C 一键停止所有服务

### 步骤

```bash
# 1. 打开终端，进入项目根目录
cd /path/to/邓兆宣

# 2. 运行启动脚本
./start-mac.sh
```

> 首次运行可能需要较长时间（安装依赖 + 编译后端），请耐心等待。

### 停止服务

在终端中按 `Ctrl+C`，脚本会自动停止所有服务。

## 🪟 Windows 端启动

### 特性
- ✅ 自动检测运行环境
- ✅ 自动安装前端依赖（已安装则跳过）

```bash

mysql -u root -pab123168 < agri-bridge\docs\数据库.sql
```

或使用 Navicat / DBeaver 等工具导入 `agri-bridge/docs/数据库.sql`。

### 步骤

```batch
# 1. 双击运行
start-win.bat

# 或在 CMD 中运行
cd /d "项目根目录路径"
start-win.bat
```

### 停止服务

分别关闭后端、管理后台、用户前台的命令窗口即可。

## ❓ 常见问题

### 1. 端口被占用
- 后端 8080 端口: 修改 `agri-bridge/src/main/resources/application.yml` 中的 `server.port`
- 管理后台 3006 端口: 修改 `art-design-pro/.env` 中的 `VITE_PORT`
- 用户前台 5173 端口: Vite 会自动递增端口号

### 2. MySQL 连接失败
- 确认 MySQL 服务已启动
- 确认密码为 `ab123168`，若不同请修改 `application.yml`

### 3. 后端编译失败
- 确认 JDK 17+ 已安装: `java -version`
- 首次运行需要下载 Maven 依赖，请确保网络畅通

### 4. 前端依赖安装失败
- 确认 Node.js 20+: `node --version`
- 确认 pnpm: `pnpm --version`
- 若失败可手动进入目录执行 `pnpm install`
