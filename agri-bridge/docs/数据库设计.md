# 农产品产销对接管理系统 - 数据库设计说明

## 说明
- 数据库：MySQL 8.x
- 命名规范：`sys_` 系统表，`dict_` 字典表，`biz_` 业务表，`msg_` 消息表
- 软删字段：`deleted`（0=未删，1=已删）
- 通用字段：`id, created_at, updated_at, created_by, updated_by, deleted`

## 表结构

### Table: sys_user
- Purpose: 用户与身份角色（采购方/生产方/管理员）
- Primary key: id (bigint, auto_increment)
- Columns:
  - username varchar(64) not null unique 用户名
  - password varchar(255) not null 密码哈希
  - role tinyint not null 角色(1管理员 2生产方 3采购方)
  - status tinyint not null default 1 状态(1启用 2禁用)
  - real_name varchar(64) null 真实姓名
  - avatar varchar(255) null 头像
  - gender tinyint null 性别(0未知1男2女)
  - address varchar(255) null 地址
- Indexes:
  - uk_user_username(username)
  - idx_user_role(role)
- Notes: 不允许双重身份

### Table: sys_qualification
- Purpose: 用户资质认证与审核
- Columns:
  - user_id bigint not null
  - type varchar(64) not null 资质类型
  - number varchar(128) not null 资质编号
  - file_url varchar(255) null 资质文件
  - status tinyint not null default 1 审核状态(1待审2通过3驳回)
  - remark varchar(255) null 审核意见
  - audit_by bigint null 审核人
  - audit_time datetime null 审核时间
- Relations:
  - user_id -> sys_user.id

### Table: dict_category
- Purpose: 农产品分类
- Columns:
  - name varchar(64) not null
  - parent_id bigint null
  - sort_no int not null default 0

### Table: dict_unit
- Purpose: 计量单位
- Columns:
  - name varchar(32) not null
  - symbol varchar(16) null

### Table: biz_product
- Purpose: 农产品档案
- Columns:
  - name varchar(128) not null
  - category_id bigint not null
  - unit_id bigint not null
  - origin varchar(128) null 产地
  - description text null
  - image_url varchar(255) null
  - traceable tinyint not null default 1 是否可溯源
  - status tinyint not null default 1 状态(1上架2下架)
  - owner_id bigint not null 生产方
- Relations:
  - category_id -> dict_category.id
  - unit_id -> dict_unit.id
  - owner_id -> sys_user.id

### Table: biz_listing
- Purpose: 信息发布（供给/求购）
- Columns:
  - type tinyint not null 类型(1供给2求购)
  - title varchar(128) not null
  - product_id bigint null
  - quantity decimal(12,2) not null
  - unit_id bigint not null
  - price decimal(12,2) null
  - location varchar(128) null
  - description text null
  - contact_name varchar(64) null
  - contact_phone varchar(32) null
  - status tinyint not null default 1 状态(1待审2通过3驳回4上架5下架6过期)
  - publisher_id bigint not null 发布人
  - start_time datetime null
  - end_time datetime null
- Relations:
  - product_id -> biz_product.id
  - unit_id -> dict_unit.id
  - publisher_id -> sys_user.id

### Table: biz_inventory
- Purpose: 库存管理
- Columns:
  - producer_id bigint not null
  - product_id bigint not null
  - quantity decimal(12,2) not null
  - unit_id bigint not null
  - location varchar(128) null
- Relations:
  - producer_id -> sys_user.id
  - product_id -> biz_product.id
  - unit_id -> dict_unit.id

### Table: biz_order
- Purpose: 订单主表
- Columns:
  - order_no varchar(32) not null unique
  - buyer_id bigint not null
  - seller_id bigint not null
  - status tinyint not null 订单状态
  - total_amount decimal(12,2) not null
  - total_quantity decimal(12,2) not null
  - delivery_address varchar(255) not null
  - contact_name varchar(64) null
  - expected_delivery_time datetime null
  - remark varchar(255) null
- Relations:
  - buyer_id -> sys_user.id
  - seller_id -> sys_user.id

### Table: biz_order_item
- Purpose: 订单明细
- Columns:
  - order_id bigint not null
  - product_id bigint not null
  - product_name varchar(128) not null
  - quantity decimal(12,2) not null
  - unit_id bigint not null
  - price decimal(12,2) not null
  - amount decimal(12,2) not null
- Relations:
  - order_id -> biz_order.id
  - product_id -> biz_product.id
  - unit_id -> dict_unit.id

### Table: biz_after_sale
- Purpose: 售后申请
- Columns:
  - order_id bigint not null
  - buyer_id bigint not null
  - seller_id bigint not null
  - type tinyint not null (1仅退款2退货退款)
  - reason varchar(255) not null
  - amount decimal(12,2) not null
  - status tinyint not null (1待审2通过3驳回4完成)
  - audit_by bigint null
  - audit_time datetime null
- Relations:
  - order_id -> biz_order.id

### Table: biz_schedule
- Purpose: 采收与配送日程
- Columns:
  - producer_id bigint not null
  - type tinyint not null (1采收2配送)
  - start_time datetime not null
  - end_time datetime null
  - location varchar(128) not null
  - route_plan text null 路线规划(前端生成)
  - order_id bigint null
- Relations:
  - producer_id -> sys_user.id
  - order_id -> biz_order.id

### Table: biz_trace_batch
- Purpose: 商品溯源批次
- Columns:
  - product_id bigint not null
  - batch_no varchar(64) not null
  - producer_id bigint not null
  - production_date date null
  - harvest_date date null
  - expiry_date date null
  - trace_info text null (JSON)
  - qr_code_url varchar(255) null
  - status tinyint not null default 1
- Relations:
  - product_id -> biz_product.id
  - producer_id -> sys_user.id

### Table: msg_conversation
- Purpose: 站内会话
- Columns:
  - user_a_id bigint not null
  - user_b_id bigint not null
  - last_message varchar(255) null
  - last_time datetime null

### Table: msg_message
- Purpose: 站内消息
- Columns:
  - conversation_id bigint not null
  - sender_id bigint not null
  - content text not null
  - type tinyint not null (1文本2图片3文件)
  - read_flag tinyint not null default 0
  - sent_at datetime not null
- Relations:
  - conversation_id -> msg_conversation.id

### Table: sys_notice
- Purpose: 公告
- Columns:
  - title varchar(128) not null
  - content text not null
  - publish_status tinyint not null (1草稿2已发布)
  - publish_time datetime null
  - publisher_id bigint null

### Table: sys_audit_record
- Purpose: 审核记录
- Columns:
  - target_type tinyint not null (1发布2资质)
  - target_id bigint not null
  - status tinyint not null (1通过2驳回)
  - remark varchar(255) null
  - auditor_id bigint not null
  - audit_time datetime not null

### Table: sys_export_record
- Purpose: 数据导出记录
- Columns:
  - exporter_id bigint not null
  - type varchar(64) not null
  - params text null
  - file_name varchar(128) null
  - file_url varchar(255) null
  - status tinyint not null default 1 (1生成中2完成3失败)
